HTML5 新增了Canvas元素，支持JavaScript脚本绘制直线、矩形、圆形等基本图形，也可以在画布上输出文字，显示图像，通过各种元素的组合，设计丰富多彩的页面。

什么是canvas：
# canvas元素是HTML5 新增专门绘制图形的元素
# canvas元素使用javaScript在网页绘制图形
# canvas元素是一个矩形区域，用户可以控制区域内的每一像素
# canvas元素拥有多种绘制路径，矩形圆形、字符以及添加图像的方法

如何使用canvas绘制图像：
1. 创建canvas元素
在html5 代码中添加canvas元素，并规定元素的id、宽度和高度。
<canvas id="myCanvas" width="300" height="150"></canvas>

2. 获取canvas元素，通过javascript根据id获取canvas元素。
var c = document.getElementById("myCanvas");

3. 获取上下文。通过getContext方法获取上下文，创建context对象
var cxt = c.getContext("2d");

4. 绘制图形。例如，用一个红色填充的矩形的代码如下：
cxt.fillSTYle = "red";
cxt.fillRect(10,10,100,50);
